# nvim-lua-configuration
Personal Neovim configuration I use. 
<br>
A fork of [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim) that I used to learn about Neovim and edited down to fit my specific needs

## NVIM Installation:
#### NOTE: NeoVim version must be greater than 0.10 for these to automatically work

Instructions here: https://github.com/neovim/neovim/releases/tag/stable

Create nvim/{user}/core dir in the `.config` dir:
```
mkdir -p ~/.config/nvim/
```

Clone the contents of this repository into the new config/nvim dir :
```
cd ~/.config/nvim
git clone {HTTPS/SSH}
```

Launch nvim for the first time and everything should be installed
```
nvim
```
